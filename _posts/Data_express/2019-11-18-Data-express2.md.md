#### 4.2 Matplotlib库的使用

*Matplotlib库*有各种可视化类构成，内部结构复杂，受*Matlab*启发。

`matplotlib.pyplot`是绘制各类可视化图形的命令子库，相当于快捷方式。

```python
import matplotlib.pyplot as plt
```
